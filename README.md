# Mein erster PR

Ein kleines Übungsprojekt, um den kompletten Git- und GitHub-Workflow einmal sauber durchzuspielen: Branch → Commit → Push → Pull Request.

## Funktionalität

Das Projekt enthält ein winziges Python-Skript, das eine Begrüßung ausgibt.

## Installation

Es werden keine externen Abhängigkeiten benötigt — Python 3 genügt.

## Verwendung

```bash
python greet.py
```

## Tests

```bash
python -m unittest
```
